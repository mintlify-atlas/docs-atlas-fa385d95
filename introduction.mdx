## What is AI Agent Deep Dive?

AI Agent Deep Dive is a research and teaching project that reverse-engineers the product requirements of a production-grade AI coding agent. It combines:

- **Architecture research docs** — 17 documents that reverse the product design of a real AI coding agent runtime, written from a product manager's perspective without leaking source code.
- **A minimal Python agent** — a clean, pedagogical implementation (`agt`) that demonstrates the core structure every agent needs: main loop, tool system, skills discovery, LLM interface, and CLI.

The goal is not to build a clone, but to understand *why* each design decision exists — and provide a stable skeleton for learning and further development.

## Why this project exists

Most chat models answer once and stop. Simple agents call tools but lack governance. A real engineering agent needs:

- Stable multi-turn execution
- A governed tool pipeline with permissions and hooks
- Context management that survives long tasks
- Extensible skills, plugins, and external tool integrations
- Verifiable outcomes, not self-declared completion

This project documents all of those requirements, and provides a minimal Python codebase that makes the structure concrete.

```
ai-agent-deep-dive/
├── src/agt/          # Teaching Python agent
│   ├── agent.py      # Core Agent class, Tool, Message, LLMClient
│   └── cli.py        # CLI entrypoint (agt command)
├── docs/             # Architecture research documents (Chinese)
└── pyproject.toml    # Poetry project, Python ≥ 3.11
```
